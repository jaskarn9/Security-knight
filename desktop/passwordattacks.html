    Password spraying Attacks

    Hackers gain access to used malicious actor attempts the same password on many accounts before moving on to another one and repeating the process.
    Dictionary Attacks

    Hacker systematically entering every word in a dictionary to break the password.
    Brute-force Attacks

    a hacking method that uses trial and error to crack passwords, login credentials, and encryption keys.
    Rainbow Table Attacks

    A rainbow table attack is a password cracking method that uses a special table to crack the password hashes in a database. Applications don't store passwords in plaintext, but instead encrypt passwords using hashes.
